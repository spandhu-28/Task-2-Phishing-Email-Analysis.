# Task-2-Phishing-Email-Analysis.
Analysis of a phishing email to detect spoofing, malicious links, and security risks
## Objective
To identify phishing indicators in a suspicious email.

## Steps Performed
1. Collected a sample phishing email
2. Analyzed sender address
3. Checked suspicious links
4. Identified phishing traits

## Findings
- Fake sender email
- Urgent language
- Suspicious URL
- Threat message

## Conclusion
The email is a phishing attempt designed to steal sensitive information.
